## How to Add / Replace Images

1. Save your photo with the file name shown above (or add a new name).
2. Place it in this `images/` folder.
3. The website will automatically display it in the correct location.
4. To add extra views of a product, upload the file here and add the path
   in the admin panel or in the `DEFAULT_PRODUCTS` array in `app.js`.

## Supported Formats

PNG, JPG, JPEG, SVG, and WebP are all supported.
